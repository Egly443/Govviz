# Govviz

Long-run performance data for UK government departments, presented in context
rather than in isolation. Each department has a hero trend, a set of core
"competence" measures, optional supporting context, and a synthesised
competence rating.

Built with **Vite + React + TypeScript**, **TanStack Router** (file-based
routing), **Tailwind CSS v4**, **Recharts**, and **lucide-react**.

## Departments

DHSC (Health & Social Care), DfE (Education), Home Office, MoJ (Justice),
MoD (Defence), DWP (Work & Pensions), and DfT (Transport).

> Figures are illustrative for demonstration. Series are deterministically
> generated from published anchor points (NHS England, NAO, ONS, IPA, MoJ,
> DfE, DWP, DfT and others) — see `sourceUrl` on each series in
> `src/components/data.ts` and `src/components/departments.ts`.

## Getting started

```bash
npm install
npm run dev      # start the dev server (http://localhost:5173)
npm run build    # production build to dist/
npm run preview  # preview the production build
npm run typecheck
```

## Project structure

```
index.html
src/
  main.tsx               # app entry, creates the router
  index.css              # Tailwind v4 import + theme tokens (dark)
  routeTree.gen.ts       # generated by the TanStack Router plugin
  routes/
    __root.tsx           # root layout (<Outlet />)
    index.tsx            # "/" redirects to /dhsc
    $dept.tsx            # "/$dept" renders a department dashboard
  components/
    DepartmentPage.tsx   # page layout for a department
    DepartmentTabs.tsx   # sticky department switcher
    TopNav.tsx           # top navigation bar
    TrendPanel.tsx       # the main trend chart card
    TurnoverBreakdown.tsx# workforce small-multiples (DHSC)
    departments.ts       # department registry + per-department series
    data.ts              # shared series types, generators, and helpers
```

## Routing

`/` redirects to `/dhsc`. Each department is reachable at `/<code>`, e.g.
`/dhsc`, `/dfe`, `/home-office`, `/moj`, `/mod`, `/dwp`, `/dft`. Unknown codes
render a 404 with a link back to DHSC.
